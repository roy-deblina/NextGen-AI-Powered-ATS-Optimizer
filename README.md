# 🚀 NextGen AI-Powered ATS Optimizer

An advanced AI system designed to improve resume selection rates by going beyond keyword matching and leveraging **semantic understanding using NLP, Transformers, and Retrieval-Augmented Generation (RAG).**

---

## 📌 Problem Statement

Traditional ATS (Applicant Tracking Systems) reject up to **75% of qualified resumes** due to:
- Keyword mismatches
- Formatting issues
- Lack of semantic understanding

---

## 💡 Solution

This project introduces an **AI-powered ATS optimizer** that:
- Understands contextual meaning of resumes
- Matches resumes with job descriptions semantically
- Provides actionable insights and automated improvements

---

## ⚙️ Key Features

### 🔹 Semantic Scoring Engine
- Generates ATS compatibility score (0–100%)
- Based on experience, skills, and education

### 🔹 Gap Analysis
- Identifies missing skills
- Detects weak "weasel words"
- Suggests improvements

### 🔹 AI Resume Rewriter (RAG-based)
- Generates optimized bullet points
- Context-aware suggestions using LLMs

### 🔹 Smart Feedback System
- Header standardization suggestions
- Quantification improvements
- Formatting alerts

---

## 🏗️ Architecture

### 🔸 V1: Baseline NLP Model
- TF-IDF + Cosine Similarity
- Basic keyword matching

### 🔸 V2: Advanced AI System
- Transformer-based embeddings (BERT/Gemini)
- Vector Database (Chroma/Pinecone)
- RAG Pipeline:
  - Retrieval → JD requirements
  - Generation → Resume enhancement

---

## 🧪 Datasets Used

- 📄 Resume Dataset (3,500+ samples)
- 💼 LinkedIn Job Postings (124K+)
- 🧠 SNLI Dataset (570K pairs for contrastive learning)

---

## 🧠 Methodology

- Text preprocessing & segmentation
- Named Entity Recognition (NER)
- Sentence embeddings using Siamese Transformers
- Contrastive learning (SimCSE)
- Semantic similarity scoring
